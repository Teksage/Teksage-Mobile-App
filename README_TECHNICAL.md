# AstroPrompt / Teksage – Technical Documentation

AstroPrompt (v1.0) and Teksage (v2.0) are **Flutter-based AI-powered astrology mobile applications**.  
They deliver real-time personalized astrological insights using **REST APIs** hosted on **AWS** and integrate with **AI models** for natural conversation and prediction.

---

## 🧱 Project Overview

- **App Name (v1):** AstroPrompt  
- **App Name (v2):** Teksage  
- **Frontend Framework:** Flutter  
- **Language:** Dart  
- **Backend:** Python (FastAPI)  
- **Hosting:** AWS (EC2)  
- **Database:** PostgresSQL
- **AI Integration:** Custom NLP + Astrology Logic Engine  
- **Voice Features:** Speech-to-Text (STT) & Text-to-Speech (TTS)  
- **Planetary Data Source:** Swiss Ephemeris (NASA-grade algorithms)
